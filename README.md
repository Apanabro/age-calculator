# Age Calculator

Professional age calculator with Apple-level UI/UX. Calculate your exact age in years, months, days, hours, and minutes.

## Features

- Dark glassmorphism UI with smooth animations
- Animated ring progress for age
- Years, months, days, hours, minutes breakdown
- Next birthday countdown
- Day of the week you were born
- Zodiac sign
- Total days alive
- PWA installable on Android/iOS
- Offline support
- Responsive design
- Reduced motion support

## Live Demo

🔗 [View Live](https://apanabro.github.io/age-calculator/)

## Tech Stack

- HTML5
- CSS3 (Glass Morphism, Animations)
- Vanilla JavaScript
- PWA (Service Worker + Manifest)
